## 📋 概述

本文档基于对20个实际ANSYS CDB测试文件的分析，列出所有出现的ANSYS MAPDL命令，并标注当前解析器的支持状态。

## ✅ 已支持的命令 (7个)

### 核心网格命令

| 命令 | 功能 | 支持度 | 优先级 |
|------|------|--------|--------|
| **NBLOCK** | 节点块数据 | ✅ 100% | ⭐⭐⭐ 必需 |
| **EBLOCK** | 单元块数据 | ✅ 100% | ⭐⭐⭐ 必需 |
| **CMBLOCK** | 组件块数据 | ✅ 100% | ⭐⭐ 重要 |
| **ET** | 单元类型定义 | ✅ 100% | ⭐⭐⭐ 必需 |
| **RLBLOCK** | 实常数块数据 | ✅ 100% | ⭐⭐ 重要 |
| **ETBLOCK** | 单元类型块数据 | ⚠️ 部分 | ⭐⭐ 重要 |
| **\*SET** | 参数设置 | ✅ 100% | ⭐ 可选 |

**说明**:
- ✅ NBLOCK: 完整支持包括旋转角度、变长坐标
- ✅ EBLOCK: 完整支持包括材料ID、多行单元
- ✅ CMBLOCK: 支持节点/单元组件、范围展开
- ✅ ET: 单元类型ID和名称
- ✅ RLBLOCK: 实常数集完整支持
- ⚠️ ETBLOCK: 识别但未完整解析
- ✅ \*SET: 参数名/值解析

### 2️⃣ 材料属性 (3个)

| 命令 | 功能 | 优先级 | 实现难度 | 备注 |
|------|------|--------|----------|------|
| **MPDATA** | 材料属性数据 | ⭐⭐⭐ 高 | 🟡 中 | EX, PRXY, DENS等 |
| **MPTEMP** | 材料温度表 | ⭐⭐ 中 | 🟢 低 | 温度相关属性 |

**说明**:
- MPDATA定义材料属性（弹性模量、泊松比、密度等）
- 当前只解析material_id，不解析具体属性值
- MPTEMP用于温度相关材料属性

**实现建议**:
```cpp
struct MaterialProperty {
    int material_id;
    std::string property_name;  // EX, PRXY, DENS, etc.
    std::vector<double> values;
    std::vector<double> temps;   // For MPTEMP
};
```

**工作量估算**: MPDATA (1小时), MPTEMP (30分钟)

### 已支持命令详解

#### NBLOCK - 节点块数据
```
格式: NBLOCK,<fields>,<format>,<numnp>,<numnp>
示例: NBLOCK,6,SOLID,       321,       321
      (1i9,6e20.13)
      1  0.00000000E+00  0.00000000E+00  0.00000000E+00
      ...
      -1

支持功能:
✅ 节点ID
✅ 坐标 (x, y, z)
✅ 旋转角度 (可选)
✅ 坐标系统
✅ 科学计数法
✅ 固定宽度格式
✅ 变长坐标
```

#### EBLOCK - 单元块数据
```
格式: EBLOCK,<format>,<solid>,<numelem>,<numelem>
示例: EBLOCK,19,SOLID,        40,        40
      (19i10)
      1  1  1  1  0  0  0  0 20  0  1  ...
      ...
      -1

支持功能:
✅ 单元ID
✅ 材料ID ⭐
✅ 单元类型ID ⭐
✅ 实常数ID ⭐
✅ 截面ID ⭐
✅ 坐标系统
✅ 节点连接性
✅ 多行单元
✅ 变长节点列表
```

#### CMBLOCK - 组件块数据
```
格式: CMBLOCK,<name>,<entity>,<numitems>,<numitems>
示例: CMBLOCK,MY_NODES,NODE,        100,        100
      (8i10)
      1  2  3  4  5  ...
      -1

支持功能:
✅ 组件名称
✅ 实体类型 (NODE/ELEM)
✅ 成员列表
✅ 范围展开 (1:100:2)
```

#### ET - 单元类型定义
```
格式: ET,<itype>,<ename>
示例: ET,1,SOLID185

支持功能:
✅ 单元类型ID
✅ 单元类型名称
```

#### RLBLOCK - 实常数块数据

```
格式: RLBLOCK,<nsets>,<maxsets>,<maxitems>,<maxitems>

      (2i10,6g16.9)

```

### 未支持命令详解（重要命令）

#### D - 位移约束
```
格式: D,NODE,Lab,VALUE,VALUE2,NEND,NINC,KEXPND
示例: D,1,UX,0.0        ! 节点1在X方向固定
      D,ALL,UY,0.0      ! 所有节点在Y方向固定
      D,10,20,UZ,0.0    ! 节点10-20在Z方向固定

参数:
- NODE: 节点ID或节点组
- Lab: 自由度标记 (UX, UY, UZ, ROTX, ROTY, ROTZ, ALL等)
- VALUE: 约束值
- VALUE2: 第二个值 (对于复数约束)
- NEND: 结束节点
- NINC: 节点增量

实现建议:
struct DisplacementBC {
    int node_id;
    std::string dof;    // UX, UY, UZ, ROTX, ROTY, ROTZ
    double value;
};
```

#### F - 节点力
```
格式: F,NODE,Lab,VALUE,VALUE2
示例: F,10,FX,1000.0    ! 节点10施加X方向1000力
      F,20,FY,-500.0    ! 节点20施加Y方向-500力

参数:
- NODE: 节点ID
- Lab: 力方向 (FX, FY, FZ, MX, MY, MZ)
- VALUE: 力大小

实现建议:
struct NodalForce {
    int node_id;
    std::string direction;  // FX, FY, FZ, MX, MY, MZ
    double value;
};
```

#### MPDATA - 材料属性
```
格式: MPDATA,Lab,MAT,C1,C2,C3,C4,C5,C6
示例: MPDATA,EX,1,2.0E11      ! 材料1弹性模量
      MPDATA,PRXY,1,0.3      ! 材料1泊松比
      MPDATA,DENS,1,7850     ! 材料1密度

常用Lab:
- EX, EY, EZ: 弹性模量
- PRXY, PRYZ, PRXZ: 泊松比
- DENS: 密度
- GXY, GYZ, GXZ: 剪切模量
- ALPX, ALPY, ALPZ: 热膨胀系数
- C: 阻尼系数
- KXX, KYY, KZZ: 导热系数

实现建议:
struct MaterialProperty {
    int material_id;
    std::string property;   // EX, PRXY, DENS, etc.
    std::vector<double> values;
    std::vector<double> temps; // From MPTEMP
};
```
